# Todo List App

A fully functional Todo List application built with vanilla JavaScript. Features include adding, editing, deleting, and filtering tasks, plus custom enhancements like drag-and-drop reordering, a dark theme toggle, and a progress bar.

## Links

- **Live Demo:** https://daniilbechin.github.io/todo-list-app/
- **GitHub Repository:** https://github.com/DaniilBechin/todo-list-app
- **Code Quality:** https://www.codefactor.io/repository/github/daniilbechin/todo-list-app

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- **Progress bar** — visual indicator of completed tasks
- **Dark theme** — toggle between light and dark mode
- **Drag and drop** — reorder tasks by dragging
- Data persisted in localStorage

## Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- Elmish architecture (custom mini-framework)
- GitHub Pages for deployment

## Getting Started

# Clone the repository
git clone https://github.com/DaniilBechin/todo-list-app.git

# Navigate into the directory
